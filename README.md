## About

This is a recreation of the classic Pong game - one of the earliest arcade video games. Two players compete by controlling paddles to hit the ball back and forth. The first player to miss the ball loses a point!

## Features

- **2-Player Mode** - Play against a friend on the same keyboard
- **Score Tracking** - Keeps track of both players' scores
- **Smooth Controls** - Responsive paddle movement
- **Classic Gameplay** - Authentic Pong experience

## Getting Started

### Prerequisites

- Python 3.x installed on your system

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/lisaochieng/pong-game.git
   ```

2. Navigate to the project directory
   ```bash
   cd pong-game
   ```

3. Run the game
   ```bash
   python main.py
   ```
